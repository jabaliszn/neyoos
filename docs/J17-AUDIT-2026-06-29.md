That is the same recurring service bug we have been fixing across Part J.

### 2) Wrong permission strings
Route uses:
- `students.view`
- `students.manage`

Those are invalid in this repo.
Correct singular forms are needed.

### 3) UI prop bugs

- `<Badge variant="outline">`
